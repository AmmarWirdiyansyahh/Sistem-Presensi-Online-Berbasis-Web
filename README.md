## 📚 Deskripsi Proyek

Aplikasi presensi berbasis web untuk mendukung pencatatan kehadiran siswa secara real-time di sekolah menengah daerah terpencil. Sistem ini bertujuan meningkatkan efisiensi, transparansi, dan keakuratan pengelolaan kehadiran siswa.

---

## 🎯 Tujuan

- Meningkatkan efisiensi pencatatan kehadiran siswa.
- Mengurangi risiko kehilangan atau kesalahan data.
- Meningkatkan adopsi digital di daerah terpencil.

---

## 🚀 Fitur Utama

- **Pencatatan Kehadiran Harian**
- **Rekapitulasi Bulanan**
- **Dashboard Laporan**
- **Notifikasi via Email/WhatsApp**
- **Unduhan Laporan (PDF/Excel)**

---

## 🏛 Teknologi yang Digunakan

- **Frontend**: HTML5, CSS3, JavaScript (Vue.js)
- **Backend**: PHP 8 (Laravel Framework)
- **Database**: MySQL / MariaDB
- **Server**: Ubuntu Server 20.04 LTS
- **Web Server**: Apache / Nginx

---

## 📦 Struktur Project

- **frontend/**: Kode tampilan (user interface)
- **backend/**: Kode server-side dan API
- **microservices/**: Modul presensi siswa
- **SOP/**: Standar Operasional Prosedur penggunaan GitHub
- **SPK/**: Surat Perjanjian Kerjasama (Statement of Work)
- **Diagram/**: Diagram Alir dan Data Flow Diagram
